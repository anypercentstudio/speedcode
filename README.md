## installation

1. clone repo
2. install dependencies

    ```bash
    npm install
    ```

3. build the extension

    ```bash
    npm run build
    ```

4. load in chrome

    - chrome >> `chrome://extensions/`
    - toggle on **"developer mode"**
    - click **"Load unpacked"**
    - select the `dist` folder from the cloned repo
